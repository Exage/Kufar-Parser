# Проект: Мониторинг новых выгодных объявлений на Kufar

## Общая информация

**Язык программирования:** Python 3.9.6

**База данных:** SQL (MySQL/PostgreSQL)

**Сервис уведомлений:** Telegram Bot API

## Цель проекта

Разработать систему автоматического мониторинга объявлений на площадке Kufar. Система должна регулярно анализировать выбранные категории товаров, выявлять новые объявления с привлекательной ценой и отправлять уведомления в Telegram.

## Основной функционал

### 1. Парсинг объявлений

Система должна:

- Выполнять сбор данных с платформы Kufar.
- Работать только с заранее настроенными категориями и поисковыми запросами.
- Получать следующие данные по каждому объявлению:
  - ID объявления;
  - Название товара;
  - Цена;
  - Ссылка на объявление;
  - Дата публикации;
  - Местоположение (если доступно);
  - Дополнительные параметры товара.

### 2. Планировщик задач

- Парсер запускается автоматически каждые 15 минут.
- После запуска выполняется обход всех настроенных категорий.
- Новые данные сохраняются в базу данных.

### 3. Хранение данных

В SQL-базе данных хранится информация:

#### Таблица categories
- id
- category_name
- category_url
- min_price
- max_price
- is_active

#### Таблица products
- id
- kufar_id
- title
- price
- url
- location
- created_at
- parsed_at

#### Таблица notifications
- id
- product_id
- sent_at

### 4. Анализ новых объявлений

После получения данных система должна:

1. Проверить, существует ли объявление в базе данных.
2. Если объявление уже обработано — пропустить его.
3. Если объявление новое:
   - сохранить его в базу;
   - сравнить цену с установленными критериями;
   - определить, является ли предложение выгодным.

Критерии выгодности могут включать:
- цену ниже установленного порога;
- цену значительно ниже средней по категории;
- попадание в список отслеживаемых моделей или товаров.

### 5. Telegram-уведомления

При обнаружении нового выгодного объявления бот отправляет сообщение в Telegram.

Пример сообщения:

📦 Новое выгодное предложение

Название: iPhone 15 Pro 256GB
Цена: 2100 BYN
Локация: Минск

Ссылка:
https://www.kufar.by/...

Дата обнаружения:
01.06.2026 12:15

### 6. Защита от дублирования

Система не должна повторно отправлять уведомления по одному и тому же объявлению.

Для этого:
- используется уникальный идентификатор объявления (kufar_id);
- ведется журнал отправленных уведомлений.

## Алгоритм работы

1. Планировщик запускает задачу каждые 15 минут.
2. Парсер получает актуальные объявления из выбранных категорий Kufar.
3. Данные сравниваются с существующими записями в базе данных.
4. Новые объявления сохраняются.
5. Выполняется анализ цены.
6. При обнаружении выгодного предложения формируется уведомление.
7. Уведомление отправляется в Telegram.
8. Информация об отправке фиксируется в базе данных.

## Нефункциональные требования

- Python 3.9.6.
- SQL-база данных.
- Логирование всех ошибок и действий системы.
- Возможность добавления новых категорий без изменения кода.
- Возможность изменения интервала проверки.
- Стабильная работа в режиме 24/7.
- Возможность последующего масштабирования под другие площадки объявлений.

## Результат работы системы

Пользователь получает в Telegram уведомления о новых объявлениях на Kufar, которые соответствуют заданным условиям и имеют выгодную цену по сравнению с аналогичными предложениями.
