# Kufar-Parser

## Активация виртуального окружения (venv)

1. Перейдите в папку проекта:
```bash
cd /Users/n.gorkavchuk/Desktop/Kufar-Parser
```
2. Активируйте окружение:
```bash
source .venv/bin/activate
```
3. Проверьте версию Python:
```bash
python --version
```

Ожидаемая версия Python: `3.9.6`
